
# 🚀 Remaining Useful Life (RUL) Estimation using Machine Learning & Deep Learning

This project focuses on predicting the **Remaining Useful Life (RUL)** of aircraft engines using the **NASA C-MAPSS dataset**. It implements a full machine learning pipeline including data preprocessing, feature selection, classical ML models, and deep learning (LSTM).

---

## 📌 Project Overview

Predicting Remaining Useful Life is critical in **predictive maintenance**, especially in aviation, where engine failure can have severe consequences.

This project:

* Preprocesses engine sensor data
* Computes RUL labels
* Performs feature selection using statistical methods
* Trains multiple ML models
* Builds a deep learning LSTM model
* Compares performance across models

---

## 📂 Dataset

The project uses the **NASA C-MAPSS (Commercial Modular Aero-Propulsion System Simulation)** dataset.

Source: NASA

Key characteristics:

* Multiple engine units
* Time-series sensor data
* Operating conditions + 21 sensor measurements
* Degradation until failure

Files used:

* `train_FD001.txt` → training data
* `test_FD001.txt` → test trajectories
* `RUL_FD001.txt` → true Remaining Useful Life labels

## ⚙️ Workflow Pipeline

### 1. Data Loading & Exploration

* Load train/test datasets
* Assign meaningful column names
* Explore distributions and missing values

### 2. Data Cleaning

* Remove constant columns (zero variance features)
* Handle redundant operational settings

### 3. Feature Engineering

* Compute engine lifecycle (`life`)
* Calculate RUL:
  [
  RUL = \text{Max Cycle} - \text{Current Cycle}
  ]
* Apply RUL capping (max = 125)

### 4. Feature Selection

Backward Stepwise Regression using `statsmodels`:

* Removes features with high p-values (> 0.05)
* Retains statistically significant sensors

Final selected features include:

* Cycle information
* Temperature sensors
* Pressure sensors
* Fan/core speed metrics
* Turbine airflow indicators

### 📌 3. Random Forest Regressor

## 🧠 Deep Learning Model (LSTM)

A **stacked LSTM architecture** is used for time-series learning.

### Architecture:

* LSTM (128 units) + BatchNorm + Dropout
* LSTM (64 units)
* LSTM (32 units)
* Dense (64 units)
* Output layer (1 unit)

### Training Setup:

* Loss: Mean Squared Error (MSE)
* Optimizer: Adam
* Early stopping used
* Batch size: 64
* Epochs: up to 100
